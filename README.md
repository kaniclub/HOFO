# HorseFollowsYou

馬の同行Modです。

## 概要
- 馬から降りたあと、馬がプレイヤーに同行します。
- バニラの馬移動制限を尊重して、移動可能な場所だけマップをまたいで同行します
- 同行はトグルキーで開始 / 停止します（デフォルトでは "H" に設定されています）
- 馬に騎乗した時やホースフルートで呼んだ時に、同行を自動でONにできます
- Generic Mod Config Menu があれば設定画面から調整できます

## 主な設定
- 同行トグルキー
- ロード時の同行ON
- 騎乗 / フルートで同行ON
- 同行開始距離 / 停止距離
- 近距離 / 中距離 / 遠距離の速度倍率
- 経路再計算間隔
- 経路探索を打ち切る距離
- 打ち切り / 経路なし時の処理（待機 / 近くにワープ / 同行OFF / 移動しない）
- ワープ演出とサウンド
- 細い当たり判定
- プレイヤー貫通
